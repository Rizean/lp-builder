# LP Builder
## Setup for building LP Builder
1. yarn install

## Build
### Windows
1. yarn build:win 
### Linux
1. yarn build:linux
### MAC
1. yarn build:mac

## Use
1. Build: `lpBuilder build <buildPath> <sourcePath>`
2. Check Paths: `lpBuilder check <buildPath> <sourcePath>`
3. TODO: Watch: `lpBuilder watch <buildPath> <sourcePath>`
